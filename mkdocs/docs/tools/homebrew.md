# Homebrew

[Homebrew](https://brew.sh) is an macOS package management system.

!!! note
    Guide version: **1.3.8**

### Check the version

```bash
brew --version
```

### Find a package

```bash
brew search <package name>
```

### Install a package

```bash
brew install <package name>
```
