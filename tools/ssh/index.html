OpenSSH secure remote connection utility.

Note

Guide version: 7.4

Check the version

ssh -V

Add key pair cache

ssh-add ~/.ssh/id_rsa

List private keys in cache

ssh-add -l

List public keys in cache

ssh-add -L

Run the agent

ssh-agent -s

Kill the agent

pkill ssh-agent
